# R-IFF 7.0.0
## What is this?
Customizable Resouse/Interchange File Format reader/writer.

## Features

* Support IFF and RIFF to read/write.
* Automatically compute chunk size for easy modifying.
* Customizable with CLOS way.

### Alternatives.
